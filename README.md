Estudos em PHP
Este repositório reúne meus **estudos e exercícios de PHP** realizados durante a faculdade, organizados por capítulos e temas.  
Cada pasta contém exemplos práticos dos conceitos aprendidos em aula, com foco em lógica de programação e desenvolvimento web.

Estrutura do Repositório
| Pasta | Conteúdo |
|-------|-----------|
| `cap1_intro` | Introdução ao PHP e primeiros scripts |
| `cap2_variaveis` | Declaração e manipulação de variáveis |
| `cap3_condicionais` | Estruturas de decisão: `if`, `else`, `elseif`, `switch` |
| `cap4_loops` | Laços de repetição: `for`, `while`, `do while`, `foreach` |
| `cap5_funcoes` | Criação e uso de funções |
| `cap6_arrays` | Arrays simples e associativos, funções de array |
| `cap7_teste` | Exercícios de revisão e testes de aprendizado |

Objetivo
Praticar e consolidar os conceitos fundamentais do **PHP**, explorando desde a base da linguagem até a construção de pequenos scripts funcionais.
